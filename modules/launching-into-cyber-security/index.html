Risk → control mapping (sample) 
Risk: Ransomware on file shares; long RTO; data loss.
Controls (NIST CSF):
  IDENTIFY: asset inventory; crown-jewel classification
  PROTECT : MFA & least privilege; allow-listing; hardening
  DETECT  : suspicious file/process events; canary files
  RESPOND : isolation playbook; disable creds; comms
  RECOVER : immutable offsite backups; restore tests; staged recovery

AES-256 at rest (random IV; CBC + padding)

from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes
from Crypto.Util.Padding import pad, unpad
def encrypt(key, data):
    iv = get_random_bytes(16)
    c = AES.new(key, AES.MODE_CBC, iv)
    return iv + c.encrypt(pad(data, 16))
