Launching into Cyber Security Completed

Module Overview

Positioned the CIA triad within organisational risk. Connected CIA trade‑offs to NIST CSF and ISO/IEC 27001 so controls are justified by exposure, obligations and measurable effect. Built a reusable threat→weakness→mitigation vocabulary to reference in later modules.

Key Submissions

  • Position paper — Pragmatic control set for SMEs (MFA, segmentation, secure backup/recovery) prioritised by risk‑reduction vs effort; mapped to NIST CSF Functions.
  • Annotated glossary — Cross‑links ATT&CK behaviours to practical mitigations (credential hygiene, allow‑listing, log integrity, least privilege).

Supporting Artefacts

  • Framework cross‑walk (NIST CSF ↔ ISO/IEC 27001 ↔ CIS Controls)
  • Initial SFIA/NICE‑aligned skills matrix

Reflection

Shifted from tool‑first to risk‑first practice. Controls now stand on the strength of their effect on risk and on verification steps, not on brand or trend.
