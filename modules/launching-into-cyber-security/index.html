Risk → control mapping (sample) 
Risk: Ransomware on file shares; long RTO; data loss.
Context: SME; mixed Windows endpoints; limited ops capacity.

Controls (mapped to NIST CSF):
  IDENTIFY (ID.AM, ID.RA): asset inventory; crown‑jewel classification
  PROTECT  (PR.AC, PR.IP): MFA, least privilege; application allow‑listing
  DETECT   (DE.AE): suspicious process/file events; canary files
  RESPOND  (RS.MI): isolate hosts; disable affected creds; comms playbook
  RECOVER  (RC.RP): immutable offsite backups; restore tests; staged recovery

AES‑256 encryption at rest (random IV; CBC + padding)

from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes
from Crypto.Util.Padding import pad, unpad

def encrypt(key, data):
    iv = get_random_bytes(16)
    cipher = AES.new(key, AES.MODE_CBC, iv)
    return iv + cipher.encrypt(pad(data, 16))

def decrypt(key, blob):
    iv, ct = blob[:16], blob[16:]
    return unpad(AES.new(key, AES.MODE_CBC, iv).decrypt(ct), 16)
