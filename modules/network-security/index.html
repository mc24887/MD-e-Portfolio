Network Security Completed

Module Overview

Covered segmentation, access control and secure protocols. Used packet analysis to evidence misconfigurations and cleartext flows; recommended least‑privilege segmentation and TLS hardening.

Key Submissions

  • Packet analysis brief — Interpreted captures to find unauthenticated services and weak ciphers; proposed pragmatic mitigations (service scoping, TLS policy, credential hygiene).
  • Segmentation rationale — Defined tiers and inter‑tier ACLs; justified micro‑segmentation where value/risk warranted it.

Supporting Artefacts

  • Wireshark notes/captures documenting findings
  • Example ACLs including allow‑list patterns and logging rules
  • TLS hygiene checklist: protocol floor, cipher policy, certificate handling

Reflection

Reading the wire provided objective evidence. Least privilege became the default and logging moved from ‘nice‑to‑have’ to a requirement for verification.

Milestones

  • Harden TLS and credential handling
