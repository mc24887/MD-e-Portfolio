Module Overview

Implemented a secure content management app (SCMA) with encrypted storage and fine-grained RBAC. Quality gates (unit tests, SAST, code style, peer review) enforced remediation before merge.

Key Submissions

  • Design → code — CryptoBox (AES-based), role checks, input validation and safe error handling.
  • CI/CD hardening — SAST (Bandit/CodeQL), linting (flake8), tests (pytest), secret scanning; failure blocks release.
Selected snippets 
from cryptography.fernet import Fernet
class CryptoBox:
    def __init__(self, key: bytes): self.f = Fernet(key)
    def encrypt(self, b: bytes) -> bytes: return self.f.encrypt(b)
    def decrypt(self, t: bytes) -> bytes: return self.f.decrypt(t)

def can_publish(user): return user.role in {'editor','admin'}
